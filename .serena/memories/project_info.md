# Project Purpose
PDF Blueprint Layers Editor for network infrastructure. Enables users to annotate PDF blueprints with multiple layers of interactive objects (icons, text, images, paths).

# Tech Stack
- **Frontend**: React 19, Vite, TypeScript, Tailwind CSS, TanStack Query, pdf-lib, react-pdf, Shadcn/UI, Lucide React.
- **Backend**: Node.js, Express, Passport.js, PostgreSQL, Drizzle ORM.
- **State Management**: Split Context Pattern (DocumentContext & UIContext).
